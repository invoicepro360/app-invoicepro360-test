# Requirements
```
# Using nvm
Install nvm using https://github.com/nvm-sh/nvm#installing-and-updating

# install node
nvm install v18.20.5
nvm use v18.20.5
nvm list

node -v
v18.20.5

npm -v
10.8.2
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build

# test locally
npm install -g serve
serve -s dist
```
